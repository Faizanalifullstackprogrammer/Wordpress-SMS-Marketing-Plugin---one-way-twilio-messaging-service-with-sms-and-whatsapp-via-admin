Twilio Account Authentication

You can easily authenticate your twilio account detail by following the listed steps.

  • Click on the Twilio SMS > Twilio Settings
  • You will see 'Twilio SID' and 'Twilio Token' input fields in Twilio Settings Option.
  • Add your Twilio SID and Token then click on Save Data button

How to send Message

The whole Twilio SMS plugin is dymaically managed for the admin panel.You can easily send the message to the person you want.
I am lisiting the steps to send message with twilio.

  • Make sure you are already save the twilio authentication settings in Twilio SMS > Twilio Settings.
  • Add your Twilio number that Twilio App provide you.
  • Now, Add the number of the user you want to send message.
  • Type message you want to send.
  • If you want to send message to sms just click on SMS MESSAGE button.
    if you want to send message through Whatsapp Add your twilio watsapp number then click on SEND WHATSAPP button.
